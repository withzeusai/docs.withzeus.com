# Recruiting Agent Examples

This page contains examples of Zeus agents designed for recruiting use cases.

This agent reviews resumes and identifies qualified candidates.

## Interview Scheduling Agent

This agent coordinates interview times between candidates and hiring teams.

## Candidate Follow-up Agent
