---
title: 'Deploying On-Premi'
---

Zeus can be deployed in 4 different ways:

1. Our cloud in a multi-tenant instance
2. Our cloud in a multi-tenant instance with your own API keys
3. Our cloud in a single-tenant instance
4. Your own cloud instance


Option 1 is the default option for all customers. If you are looking to use any of the other options, please reach out to us at [sales@withzeus.com](mailto:sales@withzeus.com)
