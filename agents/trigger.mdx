# Agent Triggers

Triggers determine when your Zeus agent is activated. This page explains how to configure different types of triggers.

## Scheduled Triggers

Configure your agent to run at specific times or intervals.

## Event-Based Triggers

Activate your agent in response to specific events like form submissions or API calls.

## Manual Triggers

Allow users to manually activate your agent through a button or command.
